# 3D Platformer Game

Juego de disparos 2D hecho con Unity. Incluye físicas, probabilidad de generación y comprobación de tales generaciones

## Tecnologías
- Unity 2022.3.9f1
- C#

## Cómo ejecutarlo
1. Clona el repo
2. Abre en Unity
3. Ejecuta la escena principal
